# Echarts Demo


## 資料夾配置

```
src/
├── assets/          # 靜態資源
│   ├── images/      # 圖片
│   ├── styles/      # 樣式檔
├── components/      # 元件
├── views/           # 頁面
├── router/          # 路由配置
├── store/           # Vuex 狀態管理
├── api/             # API 請求
├── utils/           # 工具函數
└── constants/       # 常量定義
```
